Q3D是一个在线运行的3D机械设计软件。采用Three.js作为3D渲染方案。
操作风格类似于solidworks.但是核心思想强调的是一体化设计（零件和装配体在一个界面内进行，不单独分为零件设计和装配体设计，避免了零件设计和装配体设计的割裂）。
同时未来还将引入PCB设计，使得电路，机械协同一体化设计。
避免了之前各部分分别设计，中间产生的各种需要反复协调的事情。

产品分为前端和后端。后端主要提供未来的在线零件数据，以及用户注册等。而3D设计主要是在前端实施。

请根据该项目要求，做出初步的设计。初步的设计暂时不考虑后端的功能。主要是前端的初步方案。实现第一个里程碑，就是一个基本的界面，包含三个相互垂直的面。用户鼠标点击一个面之后，可以在这个面上实现二维绘图。为之后三维的一些操作（例如拉伸凸台等）作基础。
